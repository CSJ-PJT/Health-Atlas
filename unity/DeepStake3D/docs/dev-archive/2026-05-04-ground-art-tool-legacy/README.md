# Legacy Ground Art Tool Archive

Date: 2026-05-04

## Why This Was Archived

These files belonged to the old primitive ground-art workflow for the first screen.

That workflow generated broad ground/path/plaza shapes from gameplay anchors and a JSON config. It was useful during prototype polish, but it is no longer the active production direction.

## Current Direction

DeepStake3D first-screen production has moved to Meshy/PBR asset-based environment work:

- curated Meshy models under `Assets/DeepStake/Models/Environment/`
- generated prefabs under `Assets/DeepStake/Prefabs/Environment/`
- model registry under `Assets/Resources/Meshy/`
- PBR material slots, textures, and mapping under `Assets/Resources/PBR/`

## Reference Status

Active source no longer references `first-screen-ground-art-config`.

The archived config and tool notes are kept only for historical reference. They should not be treated as active production tooling unless explicitly restored and revalidated.
